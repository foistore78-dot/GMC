# **App Name**: Garage Music Club Manager

## Core Features:

- Public Membership Request: A public-facing page where potential members can submit their application for review, protected by a CAPTCHA or similar to prevent spam.
- Secure Member Management: An admin-only page (password protected) to view, approve, edit, and remove member profiles.
- Member Database: Store member information securely (name, contact details, membership status, etc.).
- Automated Membership Updates: Automatically update membership status (e.g., expiration reminders, renewal confirmations).

## Style Guidelines:

- Dark theme with neon accents to reflect a music club atmosphere.
- Use a bold, modern font for headings and a readable font for body text.
- Music-related icons (e.g., guitars, microphones, speakers) to enhance the visual appeal.
- Clean and intuitive layout for easy navigation on both public and admin pages.
- Subtle animations and transitions to provide a dynamic user experience.
